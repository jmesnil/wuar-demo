WildFly Up and Running - Demo

WildFly is Up and Running.

This demo application contains an endpoint that returns the reverse of an text parameter

  • /reverse returns the reverse of the default text
  • /reverse?text=No+melon,+no lemon returns the reverse of "No melon, no lemon" (which is "nomel on ,nolem oN").

More info at github.com/wildfly-up-and-running
